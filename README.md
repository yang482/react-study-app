# 학습을 위한 리액트 앱

## 초기 설정

## 사용 가능한 스크립트

### `npm start`

개발 모드로 앱을 실행한다.

### `npm run build`

운영을 위한 앱을 빌드한다.

### `npm run preview`

빌드 결과를 미리보기 한다.
